Ограничение времени
3 секунды

Ограничение памяти
256 МБ

Условие задачи
Дана строка s, состоящая из строчных букв английского алфавита. Нужно найти число троек индексов (i, j, k), таких что i < j < k и одновременно s[i] == 'a', s[j] == 'b', s[k] == 'c'.

Входные данные
В единственной строке ввода находится строка s.
1 ≤ |s| ≤ 10^5.

Выходные данные
Выведите одно целое число — количество троек (i, j, k), удовлетворяющих условию.

Примеры

Пример 1
Ввод
ab
Вывод
0

Пример 2
Ввод
abc
Вывод
1
