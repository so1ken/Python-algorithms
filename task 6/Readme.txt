Задание 6
Условие:
Найти максимальную суммарную разницу в росте сотрудников, которые знакомятся попарно.

Формат входных данных:

Первая строка: n - количество сотрудников (2 <= n <= 3 * 10^5).

Вторая строка: массив целых чисел "рост сотрудников" a1, a2, .... , an (1 <= ai <= 10^9).

Формат выходных данных:
Суммарная разница.

Пример:

Ввод	Вывод
4
1 2 4 3	 4
