Задание 3
Условие:
Определить максимальное количество различных элементов в массиве после многократного деления на 2 с округлением вниз.

Формат входных данных:

Первая строка: челое число n длина массива (1 <= n <= 2 * 10^5).

Вторая строка: массив целых чисел a длинны n (1 <= ai <= 10^9).

Формат выходных данных:
Количество различных элементов.

Пример:
Ввод	Вывод
3
4 4 2	3
