Задание 4
Условие:
Найти количество подмассивов, в которых можно выбрать три числа, образующих арифметическую прогрессию.

Формат входных данных:

Первая строка: n - длина массива (3 <= n <= 10^5).

Вторая строка: массив целых чисел A1, A2, A3, .... , An (1 <= Ai <= 10).

Формат выходных данных:
Количество подмассивов.

Пример:

Ввод	Вывод
5
1 2 3 3 2	3
