Ограничение времени
8 секунд

Ограничение памяти
1024 МБ

Условие задачи
Ваш друг Илья загадал массив A длины N: A₁, A₂, …, Aₙ. Вы хотите узнать сумму всех элементов массива S = A₁ + A₂ + … + Aₙ. Илья обещал рассказать вам Q «фактов» — каждый факт i задаётся парой индексов lᵢ, rᵢ (1 ≤ lᵢ ≤ rᵢ ≤ N) и означает, что он сообщит сумму элементов на отрезке [lᵢ, rᵢ], т.е. A_{lᵢ} + A_{lᵢ+1} + … + A_{rᵢ}.

Требуется определить:

Можно ли с помощью всех Q фактов однозначно вычислить суммарное значение S.

Если можно, то вывести минимальное количество фактов (минимальный размер подмножества из данных Q фактов), достаточное для однозначного восстановления суммы S.

Входные данные
Первая строка содержит два целых числа N и Q (1 ≤ N, Q ≤ 2·10^5) — длина массива и число фактов.
Далее следуют Q строк, в каждой — два целых числа lᵢ и rᵢ (1 ≤ lᵢ ≤ rᵢ ≤ N), описывающие i-й факт.

Выходные данные
Если по всем Q фактам можно однозначно определить S, выведите в первой строке «Yes» (без кавычек), а во второй строке — одно целое число: минимальное количество фактов, необходимое для определения S (то есть минимальный размер подмножества входных отрезков, по суммам которых S однозначно восстанавливается).
Если по всем Q фактам восстановить S нельзя, выведите единственную строку «No».

Примечание о формулировке
— Факты дают значения сумм по указанным отрезкам, т.е. линейные уравнения над неизвестными A₁…Aₙ. Под «восстановить сумму S» понимается возможность однозначно вычислить линейную комбинацию A₁+…+Aₙ на основании известных значений этих отрезковых сумм.
— Под «минимальным количеством фактов» понимается размер наименьшего подмножества из данных Q отрезков, по суммам которых S однозначно определяется.
